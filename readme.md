### About
This repository for EFishery techincal test purpose. 

There's two services:

1. auth-srv: service for auth and registration
2. fetch-srv: service for fetch data 
   

### HOW TO RUN

run `docker-compose up` and app will serve on ports:

1. auth-srv : 8999
2. fetch-srv: 8999
